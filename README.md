# 🛡️ CyberShield – Cyber Security News Dashboard

![HTML](https://img.shields.io/badge/HTML5-Frontend-orange)
![CSS](https://img.shields.io/badge/CSS3-Styling-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)
![Cybersecurity](https://img.shields.io/badge/Cybersecurity-Dashboard-green)
![Status](https://img.shields.io/badge/Status-Completed-success)

CyberShield is a modern Cyber Security News Dashboard developed as part of the **Cybrexa Cyber Security Internship Program**. The platform provides real-time cybersecurity news, threat analytics, IP intelligence, and a cybersecurity knowledge hub through an interactive and responsive dashboard.

The project focuses on combining cybersecurity awareness with modern web development practices, helping users stay informed about emerging cyber threats, vulnerabilities, and security trends.
